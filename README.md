# Power BI Dashboard Project

## Overview

This project contains a Power BI dashboard file (`DASH BOARD -2.pbix`) designed for data visualization, reporting, and business insights. The dashboard helps users analyze key metrics through interactive charts, graphs, and filters.

## File Included

* `DASH BOARD -2.pbix` – Main Power BI dashboard file.

## Features

* Interactive visualizations
* Data filtering and slicers
* KPI tracking and reporting
* Business performance insights
* User-friendly dashboard layout

## Requirements

To open and use this dashboard, install:

* urlMicrosoft Power BI Desktop[https://powerbi.microsoft.com/desktop/](https://powerbi.microsoft.com/desktop/)

## How to Use

1. Download the `.pbix` file.
2. Open the file in Power BI Desktop.
3. Refresh the dataset if required.
4. Explore the dashboard using filters and visualizations.
